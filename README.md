# CaseCheck - 案例分析与查重系统

一个基于AI的案例分析系统,支持理论识别、查重分析和创新度评估。

---

## 🚀 超简单!3步完成迁移

### 当前电脑 → 新电脑

```

┌─────────────┐         ┌─────────┐         ┌──────────────┐
│双击运行      │         │         │         │双击运行       │
│一键导出.bat  │  生成   │  U盘/   │  复制   │一键安装.bat   │
│             │  ───>   │  网盘   │  ───>   │              │
│等待10秒     │         │         │         │等待2-3分钟   │
└─────────────┘         └─────────┘         └──────────────┘
                                                    │
                                                    ▼
                                            ┌──────────────┐
                                            │双击运行       │
                                            │start_windows │
                                            │    .bat      │
                                            │              │
```

## 🎯 快速开始

### Windows用户:

```bash
# 1. 当前电脑 - 导出数据
双击 → 一键导出.bat

# 2. 复制生成的文件夹到新电脑

# 3. 新电脑 - 安装
双击 → 一键安装.bat

# 4. 启动系统
双击 → start_windows.bat

# 5. 打开浏览器
访问 http://localhost:8501
```

### Mac/Linux用户:

```bash
# 新电脑安装
python3 一键安装.py

# 启动系统
./start.sh

# 打开浏览器
访问 http://localhost:8501
```
